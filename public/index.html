Simple RESTful Send Email API

Why does this API exist?

This API was created to simplify sending low volumes of email when dealing with port blocking issues.

The following providers block outbound SMTP traffic on ports 465 and 587 to prevent spam and abuse.

  • Google Cloud Platform (GCP)
  • Amazon Web Services (AWS)
  • Microsoft Azure
  • DigitalOcean
  • Linode
  • Vultr

While this API is 100% free to use with attribution, high volumes of API requests will be throttled.

API Usage Examples
cURL
curl -X POST \
https://api.yourdomain.com/api/gmail \
  -H 'Content-Type: application/json' \
  -H 'username: your-gmail-username' \
  -H 'password: your-gmail-app-password' \
  -d '{"to":"recipient@example.com","subject":"Hello","body":"This is a test email"}'
Node.js
const axios = require('axios');

axios.post('https://api.yourdomain.com/api/gmail', {
    to: 'recipient@example.com',
    subject: 'Hello',
    body: 'This is a test email'
}, {
    headers: {
        'username': 'your-gmail-username',
        'password': 'your-gmail-app-password'
    }
}).then(response => {
    console.log('Email sent:', response.data);
}).catch(error => {
    console.error('Error:', error.response.data);
});
Python
import requests

data = {
    'to': 'recipient@example.com',
    'subject': 'Hello',
    'body': 'This is a test email'
}

headers = {
    'username': 'your-gmail-username',
    'password': 'your-gmail-app-password'
}

response = requests.post('https://api.yourdomain.com/api/gmail', json=data, headers=headers)

if response.status_code == 200:
    print('Email sent:', response.json())
else:
    print('Error:', response.json())
PHP
<?php
$ch = curl_init();

$data = json_encode(array(
    'to' => 'recipient@example.com',
    'subject' => 'Hello',
    'body' => 'This is a test email'
));

curl_setopt($ch, CURLOPT_URL, 'https://api.yourdomain.com/api/gmail');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, $data);
curl_setopt($ch, CURLOPT_POST, 1);

$headers = array();
$headers[] = 'Content-Type: application/json';
$headers[] = 'username: your-gmail-username';
$headers[] = 'password: your-gmail-app-password';
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);

$result = curl_exec($ch);
if (curl_errno($ch)) {
    echo 'Error:' . curl_error($ch);
} else {
    echo 'Response:' . $result;
}
curl_close($ch);
?>
